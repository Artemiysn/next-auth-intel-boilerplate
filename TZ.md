1. Общие требования

- Использовать Next.js (App Router).
- Использовать bun
- Общий хедер должен отображаться на всех страницах.
- Реализовать i18next для переводов (минимум ru и en).
- Адаптивность.

2. Авторизация
- Авторизация может быть захардкожена
- После успешного входа состояние авторизации должно
    сохраняться при обновлении страницы и при следующем
    посещении сайта.
- Вход на страницу /about должен быть доступен только
    авторизованным пользователям.
3. Страница “О проекте”
- Должна отображать любое демо-описание проекта.
- Достаточно 2-3 блока
4. Главная страница
- Должна отображать любое демо-описание проекта.
- Достаточно 2-3 блока
5. Хедер
- Общий для всех страниц, должен содержать:
- 5.1. Home
- 5.2. About
- 5.3. Login / Logout (в зависимости от авторизации)
- 5.4. переключатель языка i18next
отображение авторизованного email (если пользователь
вошёл)
